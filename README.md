# Multilingual Calculator (多言語対応電卓)

## 概要 (Overview)
C# (Windows Forms/WPF) で開発した、英語とミャンマー語の切り替えが可能な電卓アプリです。
(A calculator application developed in C# that supports switching between English and Myanmar languages.)

機能 (Features)
* 🌍 多言語対応 (Multilingual Support): ボタン一つで英語モードとミャンマー語モードを切り替えられます。
* 🔢 基本計算 (Basic Calculation): 加算、減算、乗算、除算。
* 💻 UIデザイン: ユーザーフレンドリーなインターフェース。

## 使用技術 (Tech Stack)
* C# (.NET)
* Visual Studio

## 工夫した点 (Key Points)
* 母国語であるミャンマー語を取り入れ、ユーザビリティを意識しました。
* 言語切り替え時のロジックを効率的に実装しました。
